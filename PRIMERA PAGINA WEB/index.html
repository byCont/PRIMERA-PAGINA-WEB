Algunas Etiquetas HTML

Las etiquetas son unos de los primeros pasos para inicar en el mundo de la programacion en HTML, estos son conceptos basicos a la hora de trabajar este lenguaje de programacion web complementado con CSS y javaScript. HTML compone la estructura de nuestra pagina web, son las instruccciones especificada al navegador para mostrar texto.

    Me define el título del documento que se muestra en un browser, la barra de título o la pestaña de una página. Solo contiene texto.

    Indica la parte del cuerpo del contenido de un documento HTML. Es una etiqueta esencial para cualquier documento ya que indica donde empieza el contenido visible del documento.

    Es la parte superior del documento HTML, es donde podremos indicar los metadatos: título del documento, hojas de estilos, javaScript, CSS…

    Es una etiqueta que nos ayuda a poder crear un enlace a una página web. El atributo principal de la etiqueta HTML es href, donde pondremos el enlace al que queremos conectar. Otro atributo muy usado es target, el cual nos sirve para indicar si el enlace se abrirá en una nueva ventana o en la misma. Indica la parte del cuerpo del contenido de un documento.

    Un elemento que es usado mayoritariamente para agrupar otros elementos y actuar como plantilla de otros controles. La etiqueta nos ayuda a estructurar el documento en secciones.

    Se utiliza para títulos dentro del contenido, van desde el 'h1' hasta 'h6'.

    Se utiliza para añadir imágenes al documento.

    El elemento ol permite definir listas o viñetas ordenadas "Ordered List", bien con numeración o alfabéticamente.

    La etiqueta ul de "unordered list" lista no ordenada. Crea una lista no ordenada.

    Se usa para colocar parrafos completos directamente en el documento.

    Esta etiqueta me permite indicar información de estilos de la página directamente desde HTML sin necesidad de recurrir a CSS, esta debe definirse como elemento en la cabecera de mi página.

    B de bold que significa negrita. Indica que el texto debe ser representado como una variable bond negrita de la tipologia que se este usando.

    Span - abarcar. Es un contenedor en línea. Sirve para aplicar estilo al texto o agrupar elementos en línea. Está definido como: Elemento especial, y por lo tanto en línea, crea una caja: En línea. Puede contener: Texto, y/o Elementos en línea.

    Como su nombre lo indica se usa para insertar contenido de audio en un documento de HTML o XHTML. este se agrego como parte de HTML 5.

    El elemento HTML canvas se puede utilizar para dibujar gráficos a través de secuencias de comandos (por lo general JavaScript ). Por ejemplo, puede usarse para dibujar gráficos, hacer composiciones de fotos o incluso realizar animaciones.

    El elemento HTML "nav" representa una sección de una página cuyo propósito es proporcionar enlaces de navegación, ya sea dentro del documento actual o a otros documentos. Ejemplos comunes de secciones de navegación son menús, tablas de contenido e índices.
